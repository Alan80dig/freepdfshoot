# FreePDFShoot

**Мгновенный приватный PDF-сканер. Без серверов. Без регистрации. Без сливов.**

---

## Как это работает

1. **Снимаешь документ** одной кнопкой.
2. **Расставляешь 4 маркера** по углам — они тянутся пальцем.
3. **Сохраняешь страницу** — программа выпрямляет перспективу и повышает контраст.
4. **Добавляешь страницы** одну за другой.
5. **Печатаешь PDF** — браузер открывает окно печати со всеми страницами.

---

## Почему это безопасно

- 🔒 **Ни один байт не покидает телефон.**
- 📱 Документы хранятся **только в памяти браузера.**
- 🚫 **Без серверов.** **Без облаков.** **Без регистраций.**
- 💨 Закрыл вкладку — всё исчезло.

---

## Как запустить

Просто открой [PDFShoot](https://alan80dig.github.io/freepdfshoot/) на телефоне или компьютере.
